sample program for loading images

usage: image file

warning, sdl2_image chokes on many lbm files, especially those produced by
deluxe paint ii enhanced 2.0.

programs using sdl2_image require these files:
sdl2.dll (for sdl2 itself)
sdl2_image.dll
libjpeg-9.dll (for jpg)
libpng16-16.dll (for png)
libtiff-5.dll (for tif)
libwebp-4.dll (for webp)
zlib1.dll (for png, tif)
