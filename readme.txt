main knowledge base storage.

ALGO
====
implementations of algorithms, meant for copypasting in programming competitions

API
===
show how to use general (included) APIs in tricky ways

CRYPTO
======
routines used in cryptography (such as md5)
general stuff like primality testing/factoring goes to algo

FILEFORMAT
==========
routines for loading/parsing existing file formats

LIB
===
contains subdirectories with projects showing how to make and use them
includes: curl, gmp, opengl/freeflut, pthread, sdl, sdl_image, sdl_net

NEHE
====
the nehe opengl tutorial

examples and reference projects showing how to do elementary things with libraries
